Aplicação criada para aplicar os conhecimentos no desenvolvimento com arquitetura MVC, utilizando SPRING BOOT, JPA e testes funcionais com o Postman por meio de WebService.

As classes forma criadas com a notação @Entity para que fossem criadas as tabelas diretamente no banco de dados após inicializar a aplicação.

Foram utilizados os métodos GetMapping, PostMapping e DeleteMapping do Spring para manipular os dados dos banco de dados por meio das entidades JPA. 

As tecnologias utilizadas foram:
- IDE Eclipse
- MySQL Workbanch
- Postman
- Spring Initializr
- Github
